# Versao de Teste - Agenda de Compras

Abra o arquivo `index.html` desta pasta em qualquer navegador.

Esta versao funciona sem backend e usa dados mockados para demonstrar:

- filtros de ocorrencias
- abertura de detalhes
- sugestao de proxima data
- tratamento de uma pendencia com criacao da proxima ocorrencia

Arquivos desta versao:

- `index.html`
- `styles.css`
- `script.js`
